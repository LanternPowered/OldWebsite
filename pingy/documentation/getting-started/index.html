Getting started

Prerequisites

  • Java 8+
  • Setup the JRE environment variables

Installation

Windows

  1. Download the latest or recommend pingy build if you haven't done so already.
  2. Create a folder where you want to install the server and move the pingy jar (pingy-**-.jar) to the folder.
  3. Rename the downloaded jar file to pingy.jar.
  4. Create a script called pingy-start.bat and put the following content in it:
  5. Run the pingy-start.bat file and close the window once the server is loaded. A configuration file should be generated.
  6. Configure the server options, the explanation of the options can be found here.
  7. Run the pingy-start.bat, your ping server should now be good to go.
