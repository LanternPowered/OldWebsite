Configuration

This section will explain what all the configuration options mean.
Option Type Description
ip ip The ip address that server should be bound to.
port number The port that server should be bound to.
use-epoll-when-available true|false Whether the server should use epoll if it’s available. If it failed to apply you will be able to debug the cause in the debug mode, see advanced launch params.
message-of-the-day string|text The message of the day (motd), is displayed under the server name. Supports the new line character \n and json text format.
legacy-message-of-the-day string The message of the day (motd), is displayed under the server name. Supports only one line, this is only used for minecraft clients using version 1.6 or older.
outdated-message string The message that is displayed in the right top corner.
outdated-message-tooltip string The tooltip that will be displayed when hovering of the outdated-message. Supports the new line character \n.
disconnect-message string|text The message that will be displayed when a player tries to join the server. Supports the new line character \n and json text format.
legacy-disconnect-message string The message that will be displayed when a player tries to join the server. This is only used for minecraft clients using version 1.6 or older.
favicon path The path of the favicon file, leave this empty if you want to disable it. The path is relative to the server root.
The text data type in the table refers to the minecraft text json format, the specification can be found on the minecraft wiki.
Faulty formatted text json may crash the client and isn't validated on the server side, so be careful.
